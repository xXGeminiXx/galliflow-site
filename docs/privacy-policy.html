Galliflow Privacy Policy

Last Updated: November 2025

Overview

Galliflow is a browser extension designed to help users track and understand their browsing time. Your privacy is our top priority - all data stays on your device.

Data Collection

What We Collect

  • Domain names: The websites you visit (e.g., github.com, twitter.com)
  • Time spent: Duration of time on each website
  • Timestamps: When sessions start and end
  • Categories: The category assigned to each domain

What We DON'T Collect

  • Full URLs or page paths
  • Search queries
  • Form inputs or passwords
  • Personally identifiable information
  • Cookies or browsing history beyond domain tracking

Data Storage

All browsing/time data is stored locally on your device using Chrome's built-in storage API (chrome.storage.local). We do not send your browsing/time data to external servers, use cloud storage, or share browsing/time data with third parties.

License check: We call Google's Chrome Web Store licensing API using chrome.identity to verify your license. This sends a Google auth token to Google only for license validation. No browsing/time data is included in that request.

Data Retention

  • By default, tracking data is retained for 180 days
  • Users can configure this retention period in settings
  • Users can export all data as JSON at any time
  • Users can clear all data at any time through settings

Permissions Explained

  • tabs: Monitors the active tab to track which websites you visit. Only domain names are recorded, not full URLs.
  • storage: Saves tracking data, settings, and progress locally on your device.
  • idle: Detects when you're away from your computer to pause time tracking automatically.
  • alarms: Runs periodic tasks like saving data and sending notifications.
  • notifications: Shows alerts for goals, achievements, and daily summaries.
  • identity: Used solely to obtain a Google auth token for Chrome Web Store license verification. No browsing/time data is sent with this token.
  • host_permissions (<all_urls>): Required to track time across all websites. No data is sent externally - this permission only enables local tracking.

User Control

  • Export: Download all data as JSON from settings
  • Clear: Permanently delete all data from settings
  • Disable: Uninstall the extension to remove all stored data
  • Configure: Adjust notifications and tracking preferences

Third-Party Services

Galliflow does not use any third-party analytics, advertising, or data collection services. The extension operates entirely offline.

Children's Privacy

Galliflow does not knowingly collect information from children under 13 years of age.

Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected in the "Last Updated" date above.
