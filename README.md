# Galliflow Site (Public Policy & Support)

This repo hosts the public privacy policy and support entry points for the Galliflow Chrome extension. It is safe to keep the main extension source private while serving required public docs via GitHub Pages.

## How to publish via GitHub Pages
1. Push this repo to GitHub (e.g., `https://github.com/xXGeminiXx/galliflow-site`).
2. In GitHub repo settings, enable **Pages** with:
   - Source: `main` branch
   - Folder: `/docs`
3. After Pages builds, your policy will be at:
   - `https://xxgeminixx.github.io/galliflow-site/privacy-policy.html`

## Links to use in the Chrome Web Store listing
